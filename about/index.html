About

Software, networks, infrastructure.

I'm Morgan Bennett, a full-stack developer based between Wakefield and Newcastle, studying Networks & Cybersecurity at Northumbria University.

What I build

I design, build and ship production software: Python web apps, REST APIs, Rust desktop tools, client websites and the infrastructure they run on. My work tends to sit where software engineering, useful interfaces and network operations meet.

I self-host my own projects on a Proxmox and Docker homelab with segmented networking, firewall rules, Cloudflare DNS and Zero Trust access, and Tailscale for secure remote administration. That means I am not only writing code; I am also responsible for deployment, monitoring, security boundaries and keeping services alive.

Current focus

I am looking for placement, graduate and freelance opportunities across software development, networking and security. The projects here show the range: Resleeve for product-focused Python rendering, Prelude for backend infrastructure, a homelab for operations practice, Lodestone for Rust tooling, and Bespoke Water Solutions for client delivery.
