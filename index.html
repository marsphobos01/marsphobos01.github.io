About

marsphobos · Wakefield & Newcastle, UK

I'm Morgan Bennett, a full-stack developer based between Wakefield and Newcastle, currently studying Networks & Cybersecurity at Northumbria University. I design, build and ship production software — web apps, REST APIs and desktop tools — and I run the infrastructure it lives on.

I work across the stack and choose the right tool for each problem: Python and Flask for backends and data, Rust for systems and desktop applications, and TypeScript with modern CSS on the front end. Just as importantly, I self-host my own projects on a Proxmox/Docker homelab — VLAN-segmented networking, firewalls, Cloudflare for DNS and Zero Trust access, and Tailscale for secure remote administration. I've completed CCNA 1 through the Cisco Networking Academy. Several of my projects are live and in active use; I'm currently seeking a placement, graduate or freelance opportunity across development, networking and security.
